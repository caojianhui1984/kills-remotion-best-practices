# 海康威视 2D 摄像头人体身高测量网页版程序

本项目提供一个面向海康威视 2D 网络摄像头的身高测量网页应用。默认 RTSP 地址为：

```text
rtsp://admin:admin@192.168.1.168:554/live/main_stream
```

## 启动方式

```bash
npm run stream:bridge
npm run dev
```

浏览器不能直接播放 RTSP，因此 `server/rtsp-bridge.mjs` 使用 FFmpeg 将 RTSP 转换为 `http://localhost:8090/stream.mjpeg`，前端再读取该 MJPEG 地址。运行前请确保本机已安装 FFmpeg，并且网页主机可以访问 `192.168.1.168:554`。

## 测量流程

1. 固定摄像头，关闭自动变焦，保证地面平整且人员站在标定位置。
2. 在测量位置放置已知高度的标定杆，输入标定杆顶端/底端 Y 像素和真实高度。
3. 输入相机俯仰角、人员距离和标定距离，页面会进行透视和俯仰修正。
4. 在实时画面中依次点击头顶和脚底，系统输出身高厘米值。

## 精度说明

单目 2D 摄像头不能在未标定场景中天然保证绝对身高精度。要达到小于 5% 的测量误差，需要满足：

- 摄像头位置、焦距、俯仰角稳定；
- 人员脚底与标定杆位于同一地面测量平面；
- 标定杆高度建议为 1.6m 至 2.0m；
- 每次测量应准确取头顶最高轮廓和脚底接地点；
- 人员距离明显变化时必须更新距离参数或使用单应性地面标定。
